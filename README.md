# Snowplow2019

### Dependecies

- sdl 1.2 
  - `sudo apt install sdl1.2`
- ubuntu 16.04
- lua5.1
  - `sudo apt install lua5.1`
  - `sudo apt install liblua5.1-0`
  - `sudo apt install liblua5.1-0-dev`
  - `sudo apt install liblua5.1-0-dbg`
  - `sudo apt install lua5.1-doc`

### building 

`sh build.sh --lib`

`sh build.sh --bin`
